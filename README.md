# AI-plays-Baagchal  
In this repository, I'll be writing an AI to play Baagchal using minimax algorithm.  
  
## Installation
``configparser`` is the only dependencies other than tkinter.

``pip install configparser``

or, alternatively:

``pip install -r requirements.txt``

## Usage
``python gui.py``
